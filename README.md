# Homebrew Linux Tap

## Available Packages

- `emacs-gtk` - Emacs built with GTK+3 and Harfbuzz
- `acl2` - Theorem prover contributed to homebrew-core
- `isabelle` - Prebuilt Isabelle package, which is theorem prover
